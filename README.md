# Shapes Parser
------------------------
## Cách chạy chương trình 
1. Bật cmd lên
2. cd đến Folder chứa những file cần chạy
3. Biên dịch file exe bằng câu lệnh "g++ *.cpp -o main"
4. Chạy file exe bằng câu lệnh ".\main"
